## Getting Started

### Prerequisites
* Node JS
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/amandaduuaarte/ToDoList-ReactJS.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. run
   ```sh
   yarn start
   ```
